# Projet Express

## Prérequis
- [Node.js](https://nodejs.org) (version 18+ recommandée)
- npm (installé avec Node.js)

Vérifiez votre installation :
```bash
node -v
npm -v
```

## Installation 
```bash
git clone https://github.com/BenAubert56/ArchiWeb.git
```

```bash
cd ArchiWeb/back
```

```bash
npm install
```

```bash
npm start
```
